** EXAMPLE **

This folder contains an example of this program.
MyTempFile.txt is the file that fits the criteria in order to be read by the program.
The demo.png is a picture that shows how the window appears, along with the charts that can be made (requires jfreechart library)
The two LOG files are the txt and xml files that can be made based on the main file that appears on the window and contains information on the taxpayer.

This program can also read xml files. 600631280.xml is an example of the xml files that can be read.
The file demo2.png shows how this file appears on the window and the tabs created when 2 or more files are opened in the program
